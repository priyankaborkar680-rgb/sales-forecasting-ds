## Sales Forecasting using Time Series

### Problem Statement
Businesses struggle to predict future sales, leading to poor planning.

### Solution
Built a time-series forecasting model using ARIMA to predict future sales.

### Dataset
Monthly sales data.

### Tech Stack
Python, Pandas, Matplotlib, ARIMA

### Output
- Sales trend visualization
- 3-month sales forecast

### Business Impact
Helps companies plan inventory and revenue strategies.

### Future Improvements
- Add seasonal models (SARIMA)
- Deploy as dashboard
